What is CTRL ANNOTATE?

CTRL ANNOTATE is a high-performance, lightweight annotation tool specifically designed for YOLO (You Only Look Once) object detection formats. Stop wasting time with bloated web tools. Run locally, tag fast.

Features:

  • Direct .txt export for YOLOv5, YOLOv8, and YOLOv11.
  • Keyboard shortcuts for maximum "Control" (CTRL).
  • Native folder selection - scans subfolders automatically.
  • Train/Val split with adjustable ratio.
  • Validation preview with letterboxing.
  • Auto-save - never lose your work.
  • Undo/Redo support.
  • No internet required.
