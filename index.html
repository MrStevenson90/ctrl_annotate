What is CTRL ANNOTATE?

CTRL ANNOTATE is a high-performance, lightweight annotation tool for YOLO object detection and SAM2 segmentation models. Stop wasting time with bloated web tools. Run locally, tag fast.

Features:

  • SAM2 Magic Select - One-click AI-powered segmentation NEW
  • Group Selection - Shift+Click to combine multiple regions NEW
  • Polygon Export - JSON format for segmentation training NEW
  • Direct .txt export for YOLOv5, YOLOv8, and YOLOv11.
  • Keyboard shortcuts for maximum "Control" (CTRL).
  • Native folder selection - scans subfolders automatically.
  • Train/Val/Test split with adjustable ratios.
  • Validation preview with letterboxing.
  • Auto-save - never lose your work.
  • Undo/Redo support.
  • WebGPU accelerated (with WASM fallback).
